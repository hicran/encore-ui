Element used to provide feedback on the progress of a workflow or action.

Static Progressbars

  • There are two states for progress bars:
    • Active progress bars have animated stripes.
    • Full progress bars are solid and static in appearance.
  • Several options for displaying text with a progress bar are available:
    • No Text
    • Percentage (not italicized)
    • Detailed Text (italicized)
  • Always try to show a percentage. Your focus should be to enter specific text, since it reveals both numeric progress and provides context for what progress is being tracked. See demo below.

Dynamic Progressbars

Progressbars can also be controlled dynamically via model values.
